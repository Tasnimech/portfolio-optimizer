# Stock Portfolio Optimizer

A Python-based portfolio optimization tool that applies Modern Portfolio Theory 
to identify the optimal asset allocation across global tech stocks.



# Project Overview

This project replicates the kind of quantitative analysis used by financial analysts 
and portfolio managers. It pulls real historical stock data, runs 10,000 random 
portfolio simulations, and identifies the allocation that maximizes risk-adjusted 
returns — a concept known as the Maximum Sharpe Ratio portfolio.



# Features

- Real Market Data — pulls live historical prices via Yahoo Finance API
- Monte Carlo Simulation — generates 10,000 random portfolio allocations
- Efficient Frontier — visualizes all possible risk/return combinations
- Optimal Portfolio — identifies the Maximum Sharpe Ratio allocation
- Historical Backtest — shows how the optimal portfolio performed over 5 years
- Interactive Charts — all visualizations built with Plotly
